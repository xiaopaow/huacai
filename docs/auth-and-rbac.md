# 登录与 RBAC 权限

## 会话安全

- 密码使用 bcrypt 哈希保存，不保存明文。
- 登录成功后生成随机会话令牌。
- 数据库只保存令牌的 SHA-256 哈希。
- 会话有效期默认 7 天。
- 员工停用或重置密码时，服务端立即删除该员工全部会话。
- 员工工作量归属由服务端会话确定，客户端提交的员工 ID 不被信任。

## 初始账号

开发环境首次启动：

- 管理员用户名：`admin`
- 设计用户名：`designer`
- 审核用户名：`reviewer`

密码通过 `.env` 中的 `INITIAL_ADMIN_PASSWORD` 和 `INITIAL_EMPLOYEE_PASSWORD` 配置。项目内默认值只用于本地首次启动，正式部署前必须替换。
